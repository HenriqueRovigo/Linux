# O que é o Linux?
O **Linux** é um Sistema Operacional de código-aberto (open-source) gratuito, e disponível em diversas distribuições (distros) diferentes, cada uma com características específicas.

---

## O que o Linux tem de vantagens?
O **Linux** é gratuito, leve, e altamente personalizável, altamente recomendado para programadores e estudantes de TI. Também é recomendado para pessoas com máquinas fracas, pois é leve e pode até "reviver" máquinas antigas por sua leveza. O Linux também tem menos ameaça de vírus que o Windows.

---

## Como baixar o Linux
Você primeiro precisa de um pen-drive, e depois baixar uma imagem ISO da distro que você escolheu, como Linux Mint, Ubuntu ou Fedora. Depois de baixar a ISO no pen-drive, você deve deixar o **pen-drive bootável**, para isso você deve usar um software como Rufus (exclusivo do Windows), BalenaEtcher ou Ventoy (ambos multi-plataforma). Após deixar o pen-drive bootável, insira ele no seu desktop ou notebook, reinicie sua máquina, e acesse a BIOS e inicie pelo pen-drive. Após isso, configure e termine de baixar seu Linux e pronto.
